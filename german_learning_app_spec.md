# 🇩🇪 German Learning App – Product & Build Specification (B1 Focus)

## 🎯 Objective

Build a premium, interactive German learning web application focused on:

- Expanding vocabulary (A2 → B1/B2)
- Understanding real-life German usage
- Improving sentence construction
- Learning how Germans actually speak in different situations

This is not a beginner app. It is designed for **A2 learners moving toward B1/B2 fluency**.

---

## 🧠 Learning Philosophy

The system should follow:

Context → Sentence → Vocabulary → Explanation → Practice

NOT:

Word → Translation → Memorization

---

## 👤 Target User

- Current level: A2
- Goal: B1/B2
- Needs:
  - Real-life conversation skills
  - Better vocabulary usage
  - Confidence in speaking
  - Understanding tone (formal vs informal)

---

## 📦 Scope (v1 – Must Build)

### 1. Vocabulary System (Core Feature)

- 400–600 high-frequency B1 words
- Grouped by situations

Each word includes:
- meaning
- examples
- conjugation
- common mistakes
- usage notes
- situation tags

---

### 2. Situation-Based Learning

10–12 real-life scenarios:
- Work / Office
- Meetings
- Emails
- Small talk
- Travel
- Healthcare
- Shopping
- Bureaucracy
- Phone calls
- Housing

Each includes:
- 20–40 sentences
- vocabulary
- tone variations

---

### 3. Sentence System

- 300–500 real German sentences
- Linked to grammar + situations

---

### 4. Sentence Builder

User can:
- reorder words
- choose forms
- type freely

System:
- validates
- explains mistakes

---

### 5. Grammar Feedback

Covers:
- word order
- verb conjugation
- articles
- cases
- prepositions

---

### 6. Vocabulary Panel

Tabs:
- Meaning
- Grammar
- Usage
- Examples
- Mistakes

---

### 7. Search

Search by:
- word
- sentence
- grammar

---

### 8. Learning Modes

- Practice
- Explore
- Review

---

### 9. Error Tracking

Track:
- mistakes
- weak areas

---

### 10. Sound Feedback

- subtle
- minimal
- non-distracting

---

## 🎨 Design

- warm colors
- serif headings
- clean layout
- no flashy UI

---

## 🏗️ Tech Stack

- React + TypeScript + Vite

---

## 🚀 Phases

Phase 1:
- vocab + sentences + builder

Phase 2:
- grammar + scenarios + search

Phase 3:
- audio + tracking

---

## ⚠️ Non-Negotiables
